# Poets United

A simple space to share poetry. Post your finished pieces, tender drafts, and small truths — anonymously or not.

## Stack

- **Next.js** — frontend and API routes
- **Supabase** — database (poems storage)
- **Tailwind CSS** — styling


## Features

- Browse poems submitted by the community
- Submit your own poem via the upload form
- Post anonymously or with your name
- Line breaks are preserved as written
